# PRAGATI.AI 🇮🇳

An AI-powered portfolio manager for Indian retail investors.

## Tech Stack
- Claude API (Multi-agent AI)
- Random Forest + K-Means + HHI (ML Models)
- Ethereum Sepolia (Blockchain audit)
- FastAPI + Vanilla JS

## Setup

### 1. Clone the repo
git clone https://github.com/YourUsername/pragati-ai.git
cd pragati-ai/pragati4

### 2. Install dependencies
pip install -r backend/requirements.txt

### 3. Set up environment variables
Copy backend/.env.example to backend/.env
Fill in your API keys

### 4. Run
python -m uvicorn backend.server:app --reload --port 8000

Open frontend/index.html in browser
